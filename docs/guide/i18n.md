## 配置方式

```ts
import { createProConfig } from 'naive-ui-pro-components'
import { i18n } from './i18n'

const { provide } = createProConfig({
  locale: key => i18n.global.t(`pro.${key}`),
})

// 在 App 的 setup 中调用
provide()
```

`locale` 的类型为 `(key: ProLocaleKey) => string`。未传入时，组件使用内部维护的中文默认文案。

## 支持的 key

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| okText | `string` | `'确定'` | 确认按钮文本 |
| cancelText | `string` | `'取消'` | 取消按钮文本 |
| totalPrefix | `string` | `'共'` | 分页总数前缀 |
| reload | `string` | `'刷新'` | 刷新按钮提示 |
| fullscreen | `string` | `'全屏'` | 全屏按钮提示 |
| reset | `string` | `'重置'` | 重置按钮文本 |
| search | `string` | `'搜索'` | 搜索按钮文本 |
| collapse | `string` | `'收起'` | 收起按钮文本 |
| expand | `string` | `'展开'` | 展开按钮文本 |
| inputPlaceholder | `string` | `'请输入{label}'` | 输入框占位符模板 |
| selectPlaceholder | `string` | `'请选择{label}'` | 选择框占位符模板 |
